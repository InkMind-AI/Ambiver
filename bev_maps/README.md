# BEV maps (user-generated)

This directory is intentionally **empty** in the git repository.

Generate bird's-eye view images from ScanNet and save them here before running evaluation.

See [docs/BEV.md](../docs/BEV.md) for the paper-based method and `scripts/generate_bev_maps.py`.

Expected naming: `{scene_id}.jpg` (e.g. `scene0000_00.jpg`).
